## Key Information

### System Requirements
- Python 3.9+
- Node.js 20+
- FFmpeg (for audio)
- Optional: NVIDIA GPU with CUDA

### Quick Start Command
```bash
# Backend
cd backend && python app.py

# Frontend (new terminal)
cd frontend && npm run dev

# Open http://localhost:3000
```

### Performance
- **Speed**: 3-4 fps (CPU), 10+ fps (GPU)
- **Quality**: Full HD (1920x1080) H.264
- **Audio**: Frame-perfect sync
